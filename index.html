Guide to Connecting a Projector

A. Using Keyboard Shortcuts (Wireless)‚ñº

This method is the fastest way to connect wirelessly.

Press the Windows key + K

On your keyboard, press the Windows + K to open the 'Connect' panel.

Select Projector

Select the name of the meeting room projector from the list that appears.

B. Using an HDMI/VGA Cable‚ñº

Connect the Cable

Connect the HDMI/VGA cable from the laptop to the corresponding port on the projector.

Select Input Source

Use the projector remote to select the input source (HDMI/VGA). If the screen does not appear, press Windows + P and select 'Duplicate'.

Connection Verification

Your laptop display will appear on the projector screen.

Important: Don't forget to turn off the projector after use to save energy!
