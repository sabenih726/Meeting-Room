Panduan Menyambungkan Proyektor

🔹 A. Menggunakan Shortcut Keyboard (Wireless) ▼

Pastikan proyektor menyala

Tekan tombol power di remote proyektor. Tunggu 30-60 detik untuk warm-up, lampu indikator akan berubah dari merah ke hijau.

Tekan kombinasi tombol

Tekan Windows + K pada keyboard laptop Anda untuk membuka menu Connect.

Pilih nama ruang meeting

Cari dan pilih nama ruangan (contoh: "Meeting-Room-3F") dari daftar perangkat yang muncul.

Verifikasi koneksi

Layar laptop akan otomatis terduplikasi ke proyektor. Jika perlu, tekan Windows + P untuk mengatur mode tampilan.

💡
Tips: Pastikan laptop dan proyektor terhubung ke jaringan WiFi yang sama (TU MOBILE).
🔹 B. Menggunakan Kabel HDMI/VGA ▼

Siapkan dan sambungkan kabel

  • Ambil kabel HDMI atau VGA yang tersedia di ruangan
  • Sambungkan satu ujung ke port proyektor
  • Sambungkan ujung lainnya ke laptop
  • Pastikan kabel terpasang kencang di kedua ujung

Pilih input source yang benar

  • Gunakan remote proyektor, tekan tombol "Source/Input"
  • Pilih HDMI 1/2 atau VGA sesuai port yang digunakan
  • Tunggu 3-5 detik untuk deteksi otomatis

Atur mode tampilan

  • Tekan Windows + P
  • Pilih: Duplicate (sama), Extend (lebar), atau Second screen only
  • Sesuaikan resolusi di Settings → Display jika diperlukan
⚠️
Perhatian: Jangan mencabut kabel saat perangkat masih menyala untuk menghindari kerusakan port.
🔹 C. Troubleshooting Proyektor ▼

Masalah Umum & Solusi:

❌ Tidak Ada Gambar

  • Cek kabel power proyektor
  • Pastikan lens cap sudah dibuka
  • Periksa source input (HDMI/VGA)
  • Restart proyektor

⚠️ Gambar Buram

  • Adjust focus dengan remote
  • Bersihkan lensa proyektor
  • Atur keystone correction
  • Periksa resolusi laptop

Panduan Kamera Kandao Meeting Pro

🔹 Setup Awal Kamera ▼

Nyalakan kamera

Tekan dan tahan tombol power selama 3 detik. LED akan menyala biru saat kamera aktif.

Sambungkan ke laptop

Gunakan kabel USB 3.0 (warna biru) dari port "USB OUT" kamera ke laptop.

Sambungkan ke proyektor

Hubungkan kabel HDMI dari port "HDMI OUT" kamera ke proyektor untuk preview.

Koneksi WiFi

Settings → Network → Pilih "TU MOBILE" → Masukkan password

Test di aplikasi

Buka Zoom/Teams → Settings → Video → Pilih "Kandao Meeting Pro"

FAQ & Troubleshooting

🔧 Pertanyaan Umum

❌ Proyektor tidak terdeteksi di Windows + K? ▼

Cek koneksi WiFi

Pastikan laptop terhubung ke jaringan "TU MOBILE" yang sama dengan proyektor

Enable wireless display

Settings → System → Display → Connect to wireless display → Enable

Restart wireless adapter

Device Manager → Network adapters → Disable & Enable WiFi adapter

Update driver

Windows Update → Check for updates → Install display & network drivers

❌ Kamera tidak muncul di Zoom/Teams? ▼

Checklist Troubleshooting:

  1. Pastikan kamera sudah menyala (LED biru)
  2. Gunakan kabel USB 3.0 (port biru)
  3. Windows Settings → Privacy → Camera → Allow access ON
  4. Zoom/Teams → Settings → Video → Select "Kandao Meeting Pro"
  5. Tutup aplikasi lain yang menggunakan kamera
  6. Restart aplikasi meeting
💡
Test kamera di Windows Camera app terlebih dahulu untuk memastikan driver terinstall.
❌ Audio tidak terdengar atau terputus-putus? ▼

🔇 Check Mute Status

  • LED merah = muted
  • Tekan tombol mute di remote
  • Check app mute button

🔊 Audio Settings

  • Set Kandao as default mic
  • Disable audio enhancements
  • Check sample rate: 48kHz

🔌 Connection

  • Use USB 3.0 port
  • Avoid USB hubs
  • Update USB drivers
